* Dot product

In mathematics, the dot product is an algebraic operation that takes two equal-length sequences of numbers (usually coordinate vectors), and returns a single number.

#+begin_src
α = a · b = ⟨ a, b ⟩ = a T b = Σ aᵢbᵢ = a₁b₁ + a₂b₂ + ... + aₙbₙ
#+end_src

** Example for vectors:

#+begin_src
     1            2
     0            8
v = (2).    w = (-6).
     5            1
    -2            0

v T w = 1*2 + 0*8 + 2*(-6) + 5*1 + (-2)*0
      = 2 + 0 - 12 + 5 + 0
      = -5
#+end_src

** Example for matrices:

#+begin_src
 0  3  2.    1  0  6
-3 -3  1     2 -1. 0
 1  0  2.    5. 1. 4

v T w = 0*1 + 3*0 + 2*6 + (-3)*2 + (-3)*(-1) + 1*0 + 1*5 + 0*1 + 2*4
      = 0 + 0 + 12 - 6 + 3 + 0 + 5 + 0 + 8
      = 22
#+end_src

** Interpretation of the dot product

A single number that reflects the commonalities between two objects (vectors, matrices, tensors, signals, images, etc.). The larger the dot product, the more similar the two objects are. The smaller the dot product, the less similar they are. If the dot product is zero, then the two objects are orthogonal (i.e., they have no commonalities).
